# 🔥 4 FIRE UNITED - Official Esports Website

A premium esports team website with a complete Firebase-powered Admin CMS.

---

## 🚀 Features

- 🏠 Modern Home Page
- 👥 Dynamic Player Profiles
- 🏆 Tournament Manager
- 🖼 Gallery Manager
- ⚙️ Website Settings
- 🔐 Admin Login
- 📊 Dynamic Dashboard
- ☁️ Firebase Firestore Integration

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Firebase Firestore
- Firebase Authentication (Coming Soon)
